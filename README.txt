Jabl::RKelly is a fork of RKelly[http://rkelly.rubyforge.org/]
for parsing the dialect of Javascript used in Jabl.

Jabl::RKelly regularly merges in changes from the main RKelly branch.
The version number is kept in sync with Jabl, though, rather than with RKelly.
